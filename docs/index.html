What is Astro? ​

Astro is Neptune Fintech's commercial implementation of the OpenWave standard — a unified payment and open banking protocol for Libya.

Accept payments from any Libyan bank. Resolve customer identities via NPT alias. Access open banking data with OAuth 2.0 consent.

Payments

  • Create & manage sessions
  • NPT alias routing
  • Recurring mandates
  • Webhook events

Open Banking

  • OAuth 2.0 + PKCE consent
  • Account & balance data
  • Transaction history
  • Payment initiation

Security

  • HMAC-SHA256 webhooks
  • Short-lived session tokens
  • PKCE S256 enforced
  • Constant-time signature verify

Built on the OpenWave open standard.
