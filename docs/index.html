What is Astro? ​

Neptune. Astro is Neptune Fintech's commercial implementation of the OpenWave standard. It provides the gateway, hosted customer authentication surfaces, merchant APIs, Open Banking consent, NPT identity resolution, and operational portals.

Server firstCreate sessions, mandates, and OB consents on your backend. The browser receives only a scoped session ID or hosted URL.mk_test_ / mk_live_
Hosted SCAPayment, subscription, and Open Banking consent happen in Astro-hosted UI so merchants never collect bank OTPs directly.checkout_url
Signed outcomesUse webhooks for fulfilment. Verify signatures, enforce idempotency, then reconcile order state.X-OpenWave-Signature

  • PKCE S256 enforced
  • Constant-time signature verify

Recommended implementation sequence ​

  1. Create a merchant backend endpoint that calls POST /payments/sessions.
  2. Add the Web Drop-in or React SDK to open the hosted checkout.
  3. Register a webhook endpoint and verify X-OpenWave-Signature.
  4. Add recurring mandates only after one-time checkout is stable.
  5. Add Open Banking by creating a consent request with explicit scopes and PKCE.
